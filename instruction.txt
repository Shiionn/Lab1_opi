Исходные данные:
	1. Исходный файл библиотеки	-func.cpp
	2. Заголовочный файл 		-func.h
	3. Исходный файл программы	-test.cpp

Последовательность команд для компиляции и сборки библиотеки

	1. Переходим в папку размещения библиотеки
	2. Компиляция библиотеки
	   g++ -fPIC -c func.cpp -o func.o  /f
	3. Создание динамичемкий библиотеки
	   g++ -shared func.o -o libfunc.so
	4.Компиляция библиотеки
		!!!!УКАЗАТЬ ПУТЬ ДО ПАПКИ С ЛАБОРАТОРНОЙ ВЕЗДЕ, ГДЕ ОН НУЖЕН
	  g++ -I/media/sf_opi/lr1 -L/media/opi/lr1 test.cpp -lfunc -o test
	5. Установить и экспортировать переменную LD_LIBRARY_PATH
	   export LD_LIBRARY_PATH=/media/opi/lr1
	6. Запуск программы 
	   ./test -o join | cut | swap -f fileName /вводим одну из функций
	   Примечание. Файл ввода должен быть расположен в папке, где собран рабочий файл
